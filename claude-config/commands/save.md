---
description: Sauvegarder la session Claude Code en cours dans Obsidian/Sessions sans /exit
allowed-tools: Bash(jarvis-save-current-session.sh)
---

Exécute la commande Bash `jarvis-save-current-session.sh` pour générer un récap de la session courante (sans la fermer). Le hook recap détache en background, donc le retour est quasi-instantané.

Affiche brièvement le résultat à l'utilisateur (status + chemin du fichier attendu). N'ajoute rien d'autre — pas de re-formulation, pas de commentaire.
